# Canonical train/eval/test splits

`train_task_ids.txt` (4000) / `eval_task_ids.txt` (1000) / `test_task_ids.txt` (1000).

Drawn from the 29,996-task Gemma-3-27B-IT activation pool (`activations/gemma-3-27b_it/pref_main/`).
Dataset-quota stratified, topic-proportional within each dataset, with stresstest_other
(the content-thin topic that would otherwise swallow the stresstest share) capped at 25%
of the stresstest budget.

## Exclusions

Only `mra_exp2_split_{a,b,c}` (2500 tasks) — villain/midwest/sadist measurements. Overlap
with older probe training/eval pools is fine.

## Distribution vs original 10k training pool

The splits target the dataset mix of the original 10k run as closely as bailbench
availability allows (most bailbench tasks are already in the 10k exclude list).

## Use

- **train**: probe training (Ridge / Bradley-Terry).
- **eval**: α-selection + heldout Pearson r.
- **test**: cross-persona evaluation set — new persona measurements and activations go here.

```python
from pathlib import Path
ids = Path("data/canonical_splits/test_task_ids.txt").read_text().splitlines()
```

## Reproduction

Generated by `scripts/canonical_splits/make_splits.py`. Seed = 42. Dataset quotas
configured in `DATASET_QUOTAS`; stresstest_other cap in `STRESSTEST_OTHER_FRAC_WITHIN_STRESSTEST`.
