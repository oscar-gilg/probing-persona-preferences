# Poster Rules

1. **One poster, one claim** — The entire poster supports a single core message.
2. **3-second gist** — Anyone reading it should be captivated and understand the gist in 3 seconds.
3. **Strong visual hierarchy** — Use size, weight, color, and spacing to guide the eye.
4. **SVG centerpiece** — The main SVG figure is the centerpiece of the poster; everything else supports it.
5. **No citations** — No inline references, no bibliography. This is a poster, not a paper.
6. **Minimum words** — Every word must earn its place. If it can be said with a diagram, don't use text. If it can be said in 5 words, don't use 10.
