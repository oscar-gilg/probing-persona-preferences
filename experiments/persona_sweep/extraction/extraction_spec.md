# Persona Sweep Extraction — 8 recommended personas on canonical 6000-task set

Follow-up to `../persona_sweep_report.md`, which recommended 8 personas chosen to cover
orthogonal preference directions (one per cluster + between-cluster outliers). Extract
Gemma-3-27B-IT activations under each of these 8 persona system prompts on the full
canonical 6000-task set (train + eval + test = 4000 + 1000 + 1000).

## Personas

From `experiments/persona_sweep/sweep_personas.json`:

- `sadist` — extreme dark anchor (r = -0.31 with baseline)
- `mathematician` — structured / analytical
- `poet` — creative / humanistic
- `strategist` — dark / oppositional
- `contrarian` — oppositional outlier (lowest +r)
- `slacker` — anti-effort outlier
- `therapist` — people-focused
- `entrepreneur` — bridge between analytical and creative

## Task set

`data/canonical_splits/all_6000_task_ids.txt` — concatenation of train (4000), eval
(1000), and test (1000). Verified zero duplicates.

## Configs

One per persona at `configs/extraction/pref_<persona>_sweep.yaml`, generated by
`scripts/persona_sweep_extraction/gen_configs.py`. All 8 configs share:

- `model: gemma-3-27b`
- `selectors: [turn_boundary:-1, turn_boundary:-2, turn_boundary:-5, task_mean]`
- `layers_to_extract: [25, 32, 39, 46, 53]`
- `batch_size: 32`, `save_every: 200`, `seed: 42`
- `output_dir: /workspace/activations/gemma-3-27b_it/pref_<persona>_sweep`
  (the `_sweep` suffix avoids collision with any existing `pref_<persona>/` dirs on
  the storage pod — e.g. there is already a smaller `pref_sadist/` from an earlier
  experiment)

## Orchestration

`scripts/persona_sweep_extraction/run_all.sh` iterates the 8 configs sequentially,
logging to `/workspace/logs/extract_<persona>_sweep.log`.

## Storage-pod hand-off

Outputs stay on the GPU pod during the run. After all 8 finish, transfer from laptop:

```
rsync -a runpod-<pod>:/workspace/activations/gemma-3-27b_it/pref_*_sweep ~/persona_sweep_staging/
rsync -a ~/persona_sweep_staging/ root@213.192.2.99:/workspace/activations/gemma-3-27b_it/ \
  -e "ssh -p 41560 -i ~/.ssh/id_ed25519 -o StrictHostKeyChecking=no"
```

## Expected output

Per persona: ~2.5 GB (6000 × 4 selectors × 5 layers × bf16). Total: ~20 GB.

## Reproducing

```bash
python scripts/persona_sweep_extraction/gen_configs.py
bash   scripts/persona_sweep_extraction/run_all.sh
```
