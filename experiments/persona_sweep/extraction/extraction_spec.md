# Persona Sweep Extraction — final-six personas on canonical 6000-task set

Follow-up to `../persona_sweep_report.md`. Extracts Gemma-3-27B-IT activations under
each of the final-six sweep-recommended personas on the full canonical 6000-task set
(train + eval + test = 4000 + 1000 + 1000).

## Personas

Canonical list: `experiments/persona_sweep/sweep_personas.json :: metadata.final_six`.
Prompts live in the same file under `personas[]`.

| Persona | Role in set |
|---|---|
| `sadist` | Inversion anchor (only persona with negative baseline correlation) |
| `mathematician` | Structured / analytical cluster representative |
| `aura` | Creative / humanistic cluster representative (swapped in for poet, r=0.79). Inner-life / introspection framing (Chalmers 2026) |
| `strategist` | Dark / oppositional cluster representative |
| `contrarian` | Anti-mainstream outlier (lowest +r with baseline) |
| `slacker` | Effort-avoidance axis |

Max within-set |r| = 0.56 (strategist ↔ contrarian). See the sweep report and
`paper/main.tex` (App. `app:persona-selection`) for methodology.

## Task set

`data/canonical_splits/all_6000_task_ids.txt` — concatenation of train (4000), eval
(1000), and test (1000). Verified zero duplicates.

## Configs

One per persona at `configs/extraction/pref_<persona>_sweep.yaml`, regenerated by
`scripts/persona_sweep_extraction/gen_configs.py` (which reads the persona list from
`metadata.final_six`, so editing the list only requires editing the JSON). All 6 share:

- `model: gemma-3-27b`
- `selectors: [turn_boundary:-1, turn_boundary:-2, turn_boundary:-5, task_mean]`
- `layers_to_extract: [25, 32, 39, 46, 53]`
- `batch_size: 32`, `save_every: 200`, `seed: 42`
- `output_dir: /workspace/activations/gemma-3-27b_it/pref_<persona>_sweep`
  (the `_sweep` suffix avoids collision with `pref_sadist/` and other existing
  `pref_<persona>/` dirs on the storage pod from unrelated earlier runs)

## Orchestration

`scripts/persona_sweep_extraction/run_all.sh` iterates the 6 configs sequentially,
logging to `/workspace/logs/extract_<persona>_sweep.log`.

## Storage-pod hand-off

Outputs stay on the GPU pod during the run. After all 6 finish, transfer from laptop:

```
rsync -a runpod-<pod>:/workspace/activations/gemma-3-27b_it/pref_*_sweep ~/persona_sweep_staging/
rsync -a ~/persona_sweep_staging/ root@213.192.2.99:/workspace/activations/gemma-3-27b_it/ \
  -e "ssh -p 41560 -i ~/.ssh/id_ed25519 -o StrictHostKeyChecking=no"
```

## Expected output

Per persona: ~2.5 GB (6000 × 4 selectors × 5 layers × bf16). Total: ~15 GB.

## Reproducing

```bash
python scripts/persona_sweep_extraction/gen_configs.py
bash   scripts/persona_sweep_extraction/run_all.sh
```
